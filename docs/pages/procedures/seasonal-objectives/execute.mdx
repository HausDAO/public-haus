# Execute Seasonal Objectives Proposal

Make sure to [Distribute Retro Rewards](/procedures/retro-rewards/distribute) for the previous season before executing this proposal. Contributors should have an opportunity to claim and stake their rewards. When this proposal is executed a snapshot of members current stake is taken to capture voting weight.

To add the event to [Signals](https://signal.daohaus.fun/) the Create a Signal TCR proposal associated with the seasons retro rewards needs to be executed.

1. Goto [Proposals](https://admin.daohaus.club/#/molochv3/0xa/0xf5d6b637a9185707f52d40d452956ca49018247a/proposals)
2. View Details of the proposal
3. Click Execute
4. [Add Choices](/procedures/seasonal-objectives/add-choices)
