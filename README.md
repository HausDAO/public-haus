# PublicHaus

PublicHaus documentation bootstrapped with the [Vocs](https://vocs.dev) CLI.

Clone using the web URL.

```bash
https://github.com/HausDAO/public-haus.git
```

```bash
cd public-haus
```

```bash
pnpm install
```

```bash
pnpm dev
```

Local: [http://localhost:5173/](http://localhost:5173/)

## Deploying

```bash
pnpm build
```

- Push to GitHub
- Merge into `main`
